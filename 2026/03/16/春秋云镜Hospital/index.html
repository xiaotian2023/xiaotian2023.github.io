PostgreSQL <= 8.1 可以通过调用系统的动态链接库 libc.so.6 来实现命令执行

系统上 libc.so.6 文件的路径只能靠试（位置不对创建函数时会报错的），一般为如下几个位置：

  • /lib/x86_64-linux-gnu/libc.so.6

  • /lib/libc.so.6

  • /lib64/libc.so.6

  • /usr/lib/x86_64-linux-gnu/libc.so.6

  • /usr/lib32/libc.so.6

    创建函数成功后，执行命令时当返回值为 0 表示执行成功，其它值则是执行失败
